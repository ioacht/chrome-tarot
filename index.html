Truly Random Tarot

[click to draw a Random card]

This is a little demo page I put together to demonstrate how well the tarot can work when true randomness is involved. True as opposed to shuffling the cards yourself, where it can be argued that the arrangement is far from random. Assuming that our subconscious have far grater powers than we observe using the conscious mind, we can than explain divination and other forms for tarot use simply in terms for getting the subconscious to communicate with the conscious by fooling it to pick some cards.

On this page, the cards are picked in a process that we (humanity) currently consider to be truly random. The randomness is generated using Random.org, where it is produced from atmospheric noise.

I didn't put this page here so we can test and see if this kind of tarot usage "works", I already know that it does. I have been reading tarot like this for several years now with great success (Using my chrome browser extension).

What I want is for other people to gain some experience with it so we can discuss and theorize about the involved mechanisms.

What dose the fact that it works just as well as hand shuffling mean? Can our subconscious influence or know about atmospheric noise? Is reality much more fluid than we suspect? Is it totally created by our mind without any respect to "external" influences? Or perhaps our intuition don't really care about the card drawn and we will get to the same reading and conclusion no matter the cards we draw?
